Qpid JMS AMQP 0-x 6.4.0

Qpid JMS AMQP 0-x is JMS 1.1 compatible client which can speak AMQP 0-8,0-9,0-9-1 and 0-10.

For a detailed list of the changes in this release, see the release notes.

Download

It's important to verify the integrity of the files you download.

Qpid JMS binaries apache-qpid-jms-amqp-0-x-6.4.0-bin.tar.gz ASC, SHA512
