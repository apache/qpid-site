Qpid Broker-J 8.0.1 Release Notes

Qpid Broker-J is a message broker written in Java that stores, routes, and forwards messages using AMQP.

For more information about this release, including download links and documentation, see the release overview.

New features and improvements

  • QPID-8451 - [Broker-J][AMQP 1.0] Queues with enforced producer flow control should not allow to produce messages on sender links created on new connections
  • QPID-8454 - [Broker-J] Expose configured object statistics in prometheus format
  • QPID-8455 - [Broker-J]Use allow/deny list terminology for existing attributes and context variables
  • QPID-8458 - [Broker-J] Remove RewriteServlet from HTTP management
  • QPID-8459 - [Broker-J] Remove parsing of request URL from AnonymousInteractiveAuthenticator
  • QPID-8461 - [Broker-J] Replace Random with SecureRandom

Bugs fixed

  • QPID-8448 - [Broker-J] Http connection objects are leaked on HTTP port with a support for both TLS and TCP transports
  • QPID-8449 - [Broker-J][WMC] Query browser widget fails to load query data

Tasks

  • QPID-8464 - [Broker-J] Upgrade jackson dependencies to latest versions
  • QPID-8465 - [Broker-J] Upgrade jetty dependencies
  • QPID-8466 - [Broker-J] Upgrade test dependencies to latest versions
