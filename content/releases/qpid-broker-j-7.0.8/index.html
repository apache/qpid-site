Qpid Broker-J 7.0.8

Qpid Broker-J is a message broker written in Java that stores, routes, and forwards messages using AMQP 1.0, 0-10, 0-9-1, 0-9 and 0-8. More about Qpid.

For a detailed list of the changes in this release, see the release notes.
