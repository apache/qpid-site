Qpid Broker-J 9.1.0 Release Notes

Qpid Broker-J is a message broker written in Java that stores, routes, and forwards messages using AMQP.

For more information about this release, including download links and documentation, see the release overview.

  • QPID-8618 - [Broker-J] ACL check on link stealing
  • QPID-8620 - [Broker-J] HTTP management plugin can reveal system data or debug information
  • QPID-8621 - [Broker-J] Add operation "resetStatistics" to Producer

  • QPID-8644 - [Broker-J] JUnit 5 tests refactoring for broker-plugins/access-control
