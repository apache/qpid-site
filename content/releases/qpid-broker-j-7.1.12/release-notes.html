Qpid Broker-J 7.1.12 Release Notes

Qpid Broker-J is a message broker written in Java that stores, routes, and forwards messages using AMQP.

For more information about this release, including download links and documentation, see the release overview.

New features and improvements

  • QPID-8500 - [Broker-J] Introduce a switch to disable coalescing committer in BDB HA message store
  • QPID-8501 - [Broker-J] Upgrade bouncycastle component versions
  • QPID-8502 - [Broker-J] Upgrade jetty component version
  • QPID-8503 - [Broker-J] Upgrade Jackson version to 2.12.1
  • QPID-8505 - [Broker-J] Change data/time portion of logging string to be compatible with rfc3339
