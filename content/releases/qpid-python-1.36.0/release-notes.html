Qpid Python 1.36.0 Release Notes

Qpid Python includes an AMQP messaging library and a suite of tests for AMQP conformance.

For more information about this release, including download links and documentation, see the release overview.

Bugs fixed

  • QPID-7317 - Deadlock on publish
  • QPID-7423 - [0-8...0-91] Chunk message content that exceeds the capacity of a single frame into multiple frames
  • QPID-7424 - [0-8..0-91] Consuming python client application is not notified of remotely closed connection
  • QPID-7588 - [Python Client 0-8..0-91] The received message ocasionally might not be dispatched into the application queue in timely manner
