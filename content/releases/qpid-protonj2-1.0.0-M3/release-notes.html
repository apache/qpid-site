Qpid ProtonJ2 1.0.0-M3 Release Notes

Proton is a high-performance, lightweight messaging library. More about Qpid Proton.

For more information about this release, including download links and documentation, see the release overview.

New features and improvements

  • PROTON-2419 - [ProtonJ2] Use HTTPS where possible in project Maven pom.xml files

Bugs fixed

  • PROTON-2410 - Buffer compare uses incorrect read index for input buffer
  • PROTON-2428 - [ProtonJ2] Allow ClientOptions to be supplied without requiring container Id

Tasks

  • PROTON-2418 - [ProtonJ2] Fix tests in Client ReceiverTests not checking correct expectations
