Qpid Dispatch 1.3.0 Release Notes

Dispatch is a lightweight AMQP message router library. More about Qpid Dispatch.

For more information about this release, including download links and documentation, see the release overview.

NOTE: Dispatch 1.3.0 builds against Proton 0.25.0, but a number of tests will fail due to import errors. A patch can be found on DISPATCH-1108 to resolve these test failures until a newer release of Dispatch is available.

New features and improvements
