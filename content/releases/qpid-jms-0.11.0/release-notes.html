Qpid JMS 0.11.0 Release Notes

Qpid JMS is a complete Java Message Service 1.1 client built using the Qpid Proton protocol engine.

For more information about this release, including download links and documentation, see the release overview.

New features and improvements

  • QPIDJMS-187 - remove deprecated configuration options
  • QPIDJMS-191 - Add a WebSocket based transport to the client

Bugs fixed

  • QPIDJMS-189 - update JMSMessageID and JMSCorrelationID handling to address interop issues with non-JMS peers
  • QPIDJMS-195 - misc error handling improvements
  • QPIDJMS-196 - Anonymous Fallback producer does not handle send failure properly
  • QPIDJMS-197 - Anonymous fallback producer doesn't handle async sends correctly
  • QPIDJMS-198 - Blocked Send on a MessageProducer waiting for credit not unblocked on remote close of the producer
  • QPIDJMS-199 - Remote close of a session with producer blocked waiting for credit doesn't fail the send
  • QPIDJMS-201 - Consumer handling of no drain response can unsubscribe a durable subscription
  • QPIDJMS-204 - Race condition on Producer / Consumer create can miss remote close signal
  • QPIDJMS-205 - make shutdown of Netty executor group wait for completion

Tasks

  • QPIDJMS-200 - Update to ActiveMQ 5.14.0 release for testing
  • QPIDJMS-202 - Update to Proton 0.14.0
  • QPIDJMS-203 - Update Netty to latest bugfix release 4.0.41.Final
