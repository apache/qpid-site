Qpid C++ Broker

A message-oriented middleware message broker written in C++ that stores, routes, and forwards messages using AMQP.

Features

  • Speaks AMQP 1.0 and 0-10
  • Runs on Linux and Windows
  • Access control lists
  • Flexible logging
  • Header-based routing
  • Heartbeats
  • High availability
  • Message groups
  • Message TTLs and arrival timestamps
  • Pluggable persistence
  • Pluggable authentication via SASL
  • Producer flow control
  • Queue replication
  • Resource limits
  • Secure connection via SSL
  • Server-side selectors
  • Specialized queueing with last value queue, priority queue, and ring queue
  • Threshold alerts
  • Transactions
  • Undeliverable message handling
