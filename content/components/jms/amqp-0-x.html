Qpid JMS AMQP 0-x

A JMS 1.1 implementation supporting AMQP versions 0-10, 0-9-1, 0-9, and 0-8.

For AMQP 1.0 and JMS 2.0 / 3.0 support, use the newer Qpid JMS client.

Features

  • JMS 1.1 API
  • Secure communication via SSL and SASL
  • Producer flow control
  • Maximum delivery count
  • Message encryption
  • Failover
  • Pure-Java implementation compatible with Java 7 and above
