Qpid Proton

The AMQP messaging toolkit

Qpid Proton is a high-performance, lightweight messaging library. It can be used in the widest range of messaging applications, including brokers, client libraries, routers, bridges, proxies, and more. Proton makes it trivial to integrate with the AMQP 1.0 ecosystem from any platform, environment, or language. More about Qpid Proton.

Features

  • A flexible and capable reactive messaging API
  • Full control of AMQP 1.0 protocol semantics
  • Portable C implementation with bindings to popular languages
  • Pure-Java implementation
  • Peer-to-peer and brokered messaging
  • Secure communication via SSL and SASL
