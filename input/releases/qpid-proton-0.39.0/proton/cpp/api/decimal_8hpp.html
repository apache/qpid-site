Detailed Description

AMQP decimal types.

AMQP uses the standard IEEE 754-2008 encoding for decimal types.

This library does not provide support for decimal arithmetic, but it does provide access to the byte representation of decimal values. You can pass these values uninterpreted via AMQP, or you can use a library that supports IEEE 754-2008 and make a byte-wise copy between the real decimal values and proton::decimal values.
