Detailed Description

Client side mechanism used to select a matching mechanism from the server offered list of mechanisms. The client configures the list of allowed Mechanism names and when the server mechanisms are offered mechanism is chosen from the allowed set. If the client does not configure any mechanisms then the selector chooses from all supported SASL Mechanism types implemented in the Proton library.
