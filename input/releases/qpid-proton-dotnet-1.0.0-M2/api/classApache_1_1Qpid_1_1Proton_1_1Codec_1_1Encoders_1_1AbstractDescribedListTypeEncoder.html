virtual EncodingCodes Apache.Qpid.Proton.Codec.Encoders.AbstractDescribedListTypeEncoder< T >.GetListEncoding ( T  value)
inlineprotectedvirtual

Determine the list type the given value can be encoded to based on the number of bytes that would be needed to hold the encoded form of the resulting list entries.

Most encoders will return List32 but for cases where the type is known to be be encoded to List8 the encoder can optimize the encode step and not compute sizes.

valueThe value that is encoded as a list type
