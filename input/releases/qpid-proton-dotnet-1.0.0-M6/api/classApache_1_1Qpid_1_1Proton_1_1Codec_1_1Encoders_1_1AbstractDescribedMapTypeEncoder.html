virtual EncodingCodes Apache.Qpid.Proton.Codec.Encoders.AbstractDescribedMapTypeEncoder< K, V, M >.GetMapEncoding ( M  value)
inlineprotectedvirtual

Determine the map type the given value can be encoded to based on the number of bytes that would be needed to hold the encoded form of the resulting map entries.

Most encoders will return MAP32 but for cases where the type is known to be be encoded to MAP8 the encoder can optimize the encode step and not compute sizes.

Parameters
valueThe value that is encoded as a map type
Returns
The encoding code to use to write the map body
