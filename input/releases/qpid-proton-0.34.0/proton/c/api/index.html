Qpid Proton C API  0.34.0
Introduction

The Qpid Proton C API enables writing clients and servers that send and receive messages using the AMQP protocol. It is part of the Qpid Proton suite of messaging APIs.

Modules

The Core module is a collection of types and functions representing AMQP concepts and key elements of the API. Together they form a "protocol engine" API to create AMQP connections and links, handle events, and send and receive messages.

The Types module contains C types and functions for handling AMQP- and API-specific data types.

The Codec module has functions for AMQP data encoding and decoding.

The IO module contains interfaces for integrating with platform-native network IO. See IO integration for more information.

Conventions

Elements of the API marked as Unsettled API, including any elements contained within them, are still evolving and thus are subject to change. They are available to use, but newer versions of Proton may require changes to your application source code.

Elements marked Deprecated are slated for removal in a future release.
